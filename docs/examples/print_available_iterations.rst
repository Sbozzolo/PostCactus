print_available_iterations.py
=============================================

Given a variable, ``print_available_iterations`` prints which iterations are
available across all the data and all the dimensions. Optionally, you can
specify only one dimension. This script is useful for writing shell scripts.

Known to work with commit/release: ``1.1.0``
